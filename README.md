
# 🤖 Real-Time Hand Gesture Controlled Facial Expression System using AI

An AI-powered Computer Vision project that detects **real-time hand gestures** using a webcam and displays the corresponding **facial expression image** instantly.

This project demonstrates how gesture recognition can be used to create an interactive Human-Computer Interface (HCI) using **Python, OpenCV, and MediaPipe**.

# 📷 Output

The system performs the following steps:

1. Opens the webcam
2. Detects hand landmarks using MediaPipe
3. Recognizes the hand gesture
4. Maps the gesture to a facial expression
5. Displays the corresponding facial expression image in real time
